# 한국가스기술공사 GenOS 프로토타입

한국가스기술공사 생성형 AI 플랫폼(**GenOS**)의 관리자/사용자 페이지 UI 프로토타입입니다.
과업지시서·제안요청서 기반 요구사항을 반영하여 React + Vite + Tailwind CSS로 구현했습니다.

## 데모

배포된 데모: **https://hyegeungim3-git.github.io/kogas-genos-prototype/**

## 주요 기능

### 관리자 영역
- **관리자 대시보드** — KPI 카드 6종, 7일 이용 추이, 실시간 알림, 대기 권한 요청, 인기 에이전트 랭킹
- **사용자 및 권한 관리** — 계정 관리, 권한 요청 처리, 역할 정의
- **지식영역 관리** — RAG 지식 DB, 청크/임베딩 파이프라인 설정
- **이용 통계** — 부서·모델별 분석, 시간대별 분포, 비용 분석, ROI 메트릭
- **접근 및 감사 로그** — IP 단위 추적, 액션 필터
- **AI 품질 관리** — 전문가 검토, 신뢰도 임계값, 골든 데이터
- **공지사항** — 게시 일정, 분류, 토글 관리
- **시스템 모니터링** — 연동 SW 상태, CPU/메모리 게이지
- **보안 설정** — 2FA, IP 화이트리스트, DRM, PII 마스킹, 가드레일 룰
- **시스템 설정** — SSO/LDAP 연동, 백업 정책, 라이선스, 업데이트 이력

### 사용자 영역
- 프로필 배너 + API 사용량 게이지
- **개요** — 7일 사용량 차트, 즐겨찾는 에이전트, 최근 대화, 새 알림
- **개인 지식영역** — 다중 KA, 문서 인덱싱 상태
- **이용 기록** — 비용 요약, 대화 검색·내보내기
- **API 키** — 다중 키 발급/재발급/삭제, 권한 범위 관리
- **알림 센터** — 분류 필터, 일괄 읽음 처리
- **환경 설정** — 기본 모델/언어/테마, 알림 채널, 2FA·세션 관리

### 그 외 (좌측 사이드바)
- 대시보드 (시스템/서비스/GPU/트레이너)
- 데이터 (데이터셋·벡터 DB·자동 적재)
- 개발 (코드스페이스·공유 볼륨)
- 모델 레지스트리
- 트레이너 (LLM/VLM/임베딩/리랭킹)
- 평가 (리더보드·평가지표)
- 가드레일 (필터 설정·탐지 로그)
- 도구 (MCP 도구·서버·프롬프트 라이브러리)
- 서빙
- 에이전트 (태스크플로우 빌더/배포, 워크플로우)
- 애플리케이션 (채팅·보고서·데이터 분석)
- 운영 (승인·할당량)

## 기술 스택

- **React 19** + **Vite 7**
- **Tailwind CSS 4**
- **lucide-react** (아이콘)
- Mock 데이터 기반 — 백엔드 없이 동작

## 로컬 실행

```bash
cd genos-app
npm install
npm run dev
```

기본 포트는 `http://localhost:5173/kogas-genos-prototype/` 입니다.

## 빌드

```bash
cd genos-app
npm run build
```

산출물은 `genos-app/dist/`에 생성됩니다.

## 배포

`main` 브랜치 푸시 시 `.github/workflows/deploy.yml` 워크플로가 자동으로 빌드 후
GitHub Pages에 배포합니다.

## 디렉터리 구조

```
.
├── .github/workflows/deploy.yml    # GitHub Pages 자동 배포
├── genos-app/                      # 메인 React 앱
│   ├── src/App.jsx                 # 전체 UI (단일 파일 프로토타입)
│   └── vite.config.js
└── NanumSquareNeo-*.ttf            # 한글 폰트 (개인 사용 라이선스 확인 필요)
```
